Desarrollar un servidor que contenga los endpoints y servicios necesarios para 
gestionar los productos y carritos de compra en un e-commerce. 

Basado en Node.JS y express
Que escuche en el puerto 8080
Disponga de 2 tipos de rutas: 
/products - done
/carts - done

//////////RUTAS//////////

!! Manejo de productos con ruta /api/products/ con las siguientes rutas: 

-GET / -> lista los productos (debe de incluir ?limit) - done

GET /dc/api/products?limit=5

-GET /:pid -> producto especifico - done

-POST / -> agregar un nuevo producto con los siguientes campos obligatorios - done
excepto thumbnails: 
id (autogenerado y unico)
title : String
description : String
code : String
price : Number
status(true by default) : Boolean
stock : Number
category : String
thumbnails(de img) : Array de strings

-PUT /:pid -> tomar un producto y actualizarlo por los campos enviados desde el body. 
Nunca hay q tocar el id en esta operacion. - done

-DELETE /:pid -> elimina el producto seleccionado - done

!! Manejo de Carritos de compra con ruta /api/carts/ con las siguientes rutas: 

-GET /:cid -> lista los productos en el carrito - done

-POST / -> agregar un nuevo carrito con la siguiente estructura: 
id (autogenerado y unico)
products : Array que contenga los objetos representantes de cada producto - done

-POST /:cid/product/:pid -> agregar el producto con el siguiente formato 
al arreglo que refiere al carrito seleccionado
product : solo el id del producto
quantity : Number (en caso de que ya exista, incrementar el valor 
correspondientemente) - done

//////////PERSISTENCIA//////////

Usaremos FileSyestem con los archivos "products.json" y "carts.json" 
que respaldaran la info. - done
